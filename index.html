네이버 카페 공지 작성기

키워드를 줄마다 입력하면, 위에서부터 순서대로 톤에 맞는 공지문을 문구 전용 AI 에이전트(대화형 언어 모델)이 작성합니다. 서버와 API 키가 없을 때는 오프라인 초안으로 대체됩니다.

입력

키워드 한 줄에 하나씩
말투 (톤)

공지 만들기 시 선택한 모델로 API를 호출합니다. OpenAI 호환 엔드포인트의 배포명도 입력할 수 있습니다.

처음 로드 시 defaults/keywords-base.txt가 채워집니다. 공지·Shorts 저장 시 기본 제안 파일명은 YYYYMMDD_버튼명 형식입니다. Chrome·Edge에서는 폴더와 파일명을 고를 수 있습니다. 서버: python -m uvicorn serve:app --port 8765 (8080 충돌 시) · Ctrl+Enter로 생성 


      

      

        
        
      

      

        네이버용 복사는 서식 없는 순수 텍스트만 클립보드에 넣습니다. 카페 글쓰기에 붙여넣기(Ctrl+V)하면 본문처럼 들어가기 쉽습니다.
